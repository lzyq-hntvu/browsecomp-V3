根据查找结果，以下是 Claude Code CLI 中查看上下文使用情况的斜杠命令：

  主要命令
  ┌──────────┬────────────────────────────────────────────────────┐
  │   命令   │                        功能                        │
  ├──────────┼────────────────────────────────────────────────────┤
  │ /cost    │ 显示当前会话的 token 使用统计（输入/输出、成本）   │
  ├──────────┼────────────────────────────────────────────────────┤
  │ /stats   │ 提供使用分析模式（最全面的命令）                   │
  ├──────────┼────────────────────────────────────────────────────┤
  │ /context │ 显示上下文使用明细（系统工具、工具使用、会话数据） │
  ├──────────┼────────────────────────────────────────────────────┤
  │ /usage   │ 显示 token 消耗和当前限制                          │
  └──────────┴────────────────────────────────────────────────────┘
  推荐使用

  /stats    ← 最全面，推荐首先使用

  这个命令会显示：
  - Token 使用模式
  - 消耗趋势
  - 上下文分配情况

  注意

  2026年有用户反馈 Claude Code 2.1.1 的 token 消耗速度明显加快（4倍以上），因此监控上下文使用变得更加重要。
